<h1>Knights Travails</h1>

<p>
  A simple JavaScript solution to The Odin Project's Knights Travails project.
</p>

<h2>Features</h2>

<ul>
  <li>Checks if a board position is valid with <code>isValidPosition()</code></li>
  <li>Gets all legal knight moves with <code>getKnightMoves()</code></li>
  <li>Compares board positions with <code>samePos()</code></li>
  <li>Finds the shortest path using <code>knightMoves()</code></li>
  <li>Prints the number of moves and path with <code>printKnightMoves()</code></li>
</ul>

<h2>Technologies Used</h2>

<ul>
  <li>JavaScript</li>
  <li>Breadth-first search</li>
</ul>

<h2>Example Usage</h2>

<pre><code>printKnightMoves([3, 3], [4, 3]);

// You made it in 3 moves! Here's your path:
// [3, 3]
// [4, 5]
// [6, 4]
// [4, 3]
</code></pre>

<h2>Project Notes</h2>

<p>
  This project was built to practice graph traversal, shortest path searching,
  and problem solving with board-based movement.
</p>
